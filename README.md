# Fortune：云端记账平台

一款通过 web 能够随时访问的记账软件

> 参考教程：https://www.youtube.com/watch?v=N_uNKAus0II

环境: 

- nodejs 20

技术栈: 

- 框架：Nextjs
- 数据库：Neon+PostgreSQL
- 后端：Honojs
- 前端：React+Typescript+Tailwindcss
- 基础组件：Shadcn ui
- 图表组件：Recharts
- 鉴权：clerk
- 部署：Vercel

学习笔记：https://ge3eaqdqso.feishu.cn/docx/ZTH5duQgioXCGuxop25cvIPbnie
